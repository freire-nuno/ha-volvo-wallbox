# Terms and Conditions

This application ("Volvo Wallbox for Home Assistant") is a personal,
non-commercial Home Assistant integration that connects a user's own
Volvo-branded wallbox to their own Home Assistant instance via the official
Volvo Energy Device API.

## Scope of use

- The application is used solely by its owner to access data and controls of
  wallbox devices registered to their own Volvo ID account.
- No data is collected, stored, or shared with any third party. All data
  retrieved from the Volvo API remains within the user's own Home Assistant
  installation.
- Authentication uses the official Volvo ID OAuth2 flow; the application
  never receives or stores Volvo ID passwords.
